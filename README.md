###### На данный момент шифратор умеет:

 1) шифровать и расшифровывать сообщения с помощью шифра Цезаря, Виженера и Вернама
 
 2) взламывать зашифрованный с помощью шифра Цезаря текст методами частотного анализа
 
 3) провести частотный анализ на пользовательском тексте и выдать статистику в виде json-файла
 

###### Как запустить:
 В командной строке после вызова файла commands.py указать опции:
 
  **действие:** зашифровать (Encrypt), расшифровать(Decrypt), взломать(Hack)или получить статистику текста(Train)
  
  **шифр**
  
  **файл с текстом ключа** (цифра для шифра Цезаря, строка для шифра Виженера и Вернама)
  
  **файл с исходным текстом** (вводить как --input <имя файла>, кроме того поддерживается ввод текста в командной строке)
  
  **файл с обработанным текстом** (Вводить как --output <имя файла>. Если его не указать, то выведется в командной строке
  Для режима анализа нужно подавать на вход JSON-файл)
